# Task: Resumo de vídeos do YouTube

## Objective
Adicionar ao app a capacidade de resumir vídeos do YouTube. O usuário cola o link do vídeo, o app busca a transcrição disponível e gera um resumo com base nela — usando o mesmo pipeline de IA já existente para transcrições de áudio.

## Context
O app (Minuta) já processa transcrições de áudio local e gera resumos via LLM. A nova funcionalidade reutilizaria a etapa de resumo, mas com uma fonte diferente: a transcrição pública do YouTube no lugar de um arquivo de áudio local.

Stack: Tauri (Rust backend) + React/TypeScript frontend, com chamadas a serviços externos de transcrição/LLM.

## Requirements
- Campo para o usuário colar a URL do YouTube
- O app deve buscar a transcrição do vídeo (legendas automáticas ou manuais)
- Resumo gerado com base na transcrição, sem necessidade de baixar o áudio
- Resultado exibido na mesma interface de resumo já existente

## Constraints
- Não baixar o arquivo de áudio do vídeo — usar apenas a transcrição/legenda disponível
- Compatível com o backend Rust/Tauri existente
- Sem dependências que violem os termos de uso do YouTube de forma que inviabilize a distribuição do app

## Edge Cases
- Vídeo sem legendas disponíveis (nem automáticas nem manuais)
- Legendas disponíveis apenas em idioma diferente do desejado
- Link inválido ou vídeo privado/removido
- Transcrições muito longas que excedam o limite de contexto do modelo
